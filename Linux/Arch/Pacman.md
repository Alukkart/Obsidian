### Очистка кеша пакетов Pacman

Pacman, пакетный менеджер Arch Linux, хранит загруженные пакеты в кеше. Этот кеш может занимать много места, и его можно безопасно очистить.

#### Очистка всего кеша:
```bash
sudo pacman -Scc
```

Эта команда удалит все пакеты из кеша, включая те, которые могут быть полезны для отката или переустановки.
#### Очистка только старых версий пакетов:
```bash
sudo pacman -Sc
```

Эта команда удалит только старые версии пакетов, оставив последние версии в кеше.
